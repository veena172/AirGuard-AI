# 🌍 AirGuard AI

AirGuard AI is a prototype developed for ET AI Hackathon 2026 to help monitor and understand urban air pollution using AI.

##  About the Project

The main idea of this project is to provide a simple platform where users can view air quality information, identify possible pollution sources, and get AI-based insights for better decision-making. The project focuses on making air quality data easier to understand through an interactive dashboard.

##  Features

- Air Quality Dashboard
- AQI Monitoring
- Pollution Source Analysis
- AI-based AQI Prediction
- Interactive Charts and Visualizations
- Citizen Advisory Section

##  Technologies Used

- HTML
- CSS
- JavaScript
