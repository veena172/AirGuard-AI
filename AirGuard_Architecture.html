DATA PROCESSING & FUSION LAYER
⚡
Apache Kafka
Real-time event streaming. 50k+ events/sec throughput across all sensor feeds
🔀
Apache Spark
Distributed processing, spatial joins, sensor anomaly detection at scale
🗄️
TimescaleDB
Time-series storage optimised for sensor data. InfluxDB for high-frequency writes
🌍
PostGIS / GeoServer
Geospatial data store. Ward boundary polygons, source location registry
▼
AI INTELLIGENCE LAYER — MULTI-AGENT SYSTEM
🔬
Source Attribution Agent
Spatial-temporal ML model (XGBoost + SHAP). Attributes pollution % by source per ward with confidence scores
📈
AQI Forecast Agent
LSTM + Transformer hybrid. 24-72 hr forecast at 1km grid. RMSE 18.4 vs persistence baseline 34.2
🚨
Enforcement Intel Agent
RAG over violation history + satellite evidence. Ranks enforcement actions by impact score
💬
Citizen Advisory Agent
LLM (Claude API) generating ward-level health advisories in 12 regional languages. Personalised by vulnerability
🏭
Anomaly Detection Agent
Isolation Forest + sensor fusion. Detects unusual emission events. Alert latency <3 min
